# User Personas — TypeFlow

These user personas represent the core target archetypes for the TypeFlow platform. They help guide product decisions, UI style layouts, and functional feature prioritisation.

---

## Persona 1: Devon the Developer
*“I want to build muscle memory for programming syntax and typing symbols so I can write code as fast as I think.”*

### Profile
* **Age**: 26
* **Role**: Junior Software Engineer
* **Location**: San Francisco, CA
* **Tech Comfort**: Expert

### Goals & Motivations
* Build speed and typing accuracy on non-standard keys (brackets `{}`, brackets `[]`, operators `=>`, semicolons `;`, backticks `` ` ``).
* Ingest technical documentation (e.g., API references, markdown files) to study frameworks while practicing typing.
* Build muscle memory for keyboard-heavy environments like Vim.

### Pain Points with Existing Tools
* Traditional typing trainers only use English sentences and ignore coding-specific syntax.
* Custom lists on other platforms lack code formatting (indentation, tabs, brackets) and make code-typing practice feel unnatural.

### TypeFlow Use Case
* Ingests `.md` or `.txt` source files (e.g., React component drafts or Rust algorithm exercises).
* Focuses on custom keyboard layouts (e.g., Colemak) and custom monospace fonts to mimic his actual IDE layout.

## Persona 2: Sophia the Student
*“I want to review my biology textbook and lecture notes while practicing my typing, turning passive study time into active practice.”*

### Profile
* **Age**: 20
* **Role**: Undergraduate Biology Major
* **Location**: Boston, MA
* **Tech Comfort**: Medium-High

### Goals & Motivations
* Review study material for exams while improving her typing speed for writing essays.
* Track daily training targets to build a habit of structured typing practice.
* Earn milestones and level up to stay engaged during boring study topics.

### Pain Points with Existing Tools
* Typing trainers are childish or use content that has no relevance to her college curriculum.
* Copy-pasting short fragments of text is tedious; she wants to upload an entire study guide or chapter PDF at once.

### TypeFlow Use Case
* Uploads lecture summaries as PDFs.
* Keeps track of progress via the Dashboard widget and views her weekly calendar streaks.
* Uses the AI explanation feature (Phase 12) to clarify complex medical terms that appear in the typing box.

## Persona 3: Arthur the Aspiring Writer
*“I want to type out classic novels and my own drafts to internalize sentence structures, cadence, and vocabulary.”*

### Profile
* **Age**: 34
* **Role**: Copywriter and Novelist
* **Location**: London, UK
* **Tech Comfort**: Medium

### Goals & Motivations
* Build a typing rhythm and pacing that matches the cadence of professional literature.
* Internalize rich vocabulary and syntax by typing classic books (e.g., *Moby Dick*, *The Great Gatsby*).
* Practice typing in a clean, distraction-free environment.

### Pain Points with Existing Tools
* Typing engines focus on speed bursts rather than sustained, comfortable endurance typing.
* Visual interfaces are cluttered, flashing live metrics that disrupt creative focus.

### TypeFlow Use Case
* Uploads drafts of his own chapters to practice and refine them.
* Prefers Zen Mode, which hides timers, WPM counters, and virtual keyboards.
* Uses custom themes (e.g., Gruvbox, Nord) to create a relaxing visual environment.

## Persona 4: Layla the Language Learner
*“I want to practice typing in French and Spanish using authentic articles so I can master the layout and spelling concurrently.”*

### Profile
* **Age**: 42
* **Role**: Translator and Travel Blogger
* **Location**: Montreal, Canada
* **Tech Comfort**: Low-Medium

### Goals & Motivations
* Practice typing accents (`é`, `è`, `ü`, `ñ`, `ç`) on standard and foreign keyboard layouts.
* Expand vocabulary in target foreign languages by typing authentic news articles and literature.
* Receive visual guidance on key locations when typing on unfamiliar layouts (e.g., French AZERTY).

### Pain Points with Existing Tools
* Platforms lack support for foreign layouts and accents, flagging accented letters as errors.
* Uploading non-English text leads to weird formatting bugs and broken character splits.

### TypeFlow Use Case
* Ingests French and Spanish news articles as text files.
* Configures virtual keyboard layouts to map to the target language accent keys.
* Uses the AI translation and vocabulary builder to generate customized accent drills for keys she struggles with.
